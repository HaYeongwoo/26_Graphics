## 1. 팀과제 계획서 (제안서)

### 프로젝트명

Clean Room 3D : 스마트 청소 로봇 시뮬레이터

### 프로젝트 개요

React Three Fiber(R3F)와 Three.js를 사용해 3D 방 안에서 청소 로봇이 자동으로 이동하며 먼지를 제거하는 시뮬레이션을 구현하는 웹 기반 인터랙티브 콘텐츠이다.

사용자는 3D 공간에서 로봇의 움직임을 관찰하거나 직접 조작할 수 있으며, 바닥에 생성된 먼지를 클릭해 제거하는 인터랙션 기능을 포함한다. 단순한 구조이지만 애니메이션, 조명, 카메라 제어, 사용자 인터랙션 요소를 모두 포함하여 3D 웹 그래픽의 핵심 개념을 학습하고 구현하는 것을 목표로 한다.

### 프로젝트 목표

- React Three Fiber 기반 3D 환경 구현
- 기본 Geometry를 활용한 3D 공간 구성 능력 향상
- 애니메이션 및 사용자 인터랙션 구현
- 조명 및 카메라 제어 학습
- 실시간 객체 반응형 시스템 제작 경험 확보

### 주요 기능

**3D 공간 구성**
- 방 구조 (바닥, 벽, 천장)
- 간단한 가구 (책상, 의자 등)
- 청소 로봇
- 먼지 오브젝트

**애니메이션**
- 청소 로봇 이동
- 로봇 회전 및 방향 전환
- 먼지 제거 시 사라지는 효과 (scale/opacity 변화)

**사용자 인터랙션**
- 마우스 클릭: 먼지 제거
- 키보드 입력 : 로봇 수동 조작, 먼지 제거
- 카메라 회전 및 확대/축소: 로봇 시점

**조명 및 카메라**
- AmbientLight (전체 밝기)
- PointLight (중앙 조명)
- OrbitControls 기반 카메라 제어

**배경 및 환경**
- 실내 방 구조
- 단순한 색상 기반 텍스처
- 실내 조명 연출

### 심화 요소 적용 계획

- 파티클: 효과 먼지 제거
- GLTF 모델 로딩: 로봇 모델 사용 시
- Bloom 효과
- 텍스처 적용

### 사용 기술

- React
- Three.js
- React Three Fiber
- Drei
- JavaScript

### 기대 효과

- 3D 웹 그래픽 기본 구조 이해
- 실시간 애니메이션 구현 능력 향상
- 사용자 인터랙션 기반 UI 설계 경험
- 팀 기반 3D 프로젝트 협업 경험 확보

### 최종 목표

사용자가 3D 공간에서 청소 로봇이 움직이며 먼지를 제거하는 과정을 직관적으로 관찰하고, 직접 인터랙션할 수 있는 완성도 높은 3D 시뮬레이션 구현을 목표로 한다.

### 예상 문제 및 해결 방안

| 문제 | 해결 방법 |
|---|---|
| 로봇 이동이 부자연스러움 | 랜덤 이동 대신 순차 이동 사용 |
| 먼지 클릭 오류 | Raycaster 또는 onClick 정확도 조정 |
| 성능 저하 | 먼지 개수 제한 (20~30개) |
| 카메라 시점 문제 | OrbitControls 제한 설정 |
