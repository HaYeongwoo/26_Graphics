# 🤖 Robot Cleaner 3D

브라우저에서 동작하는 3D 로봇 청소기 시뮬레이션입니다. 방 안에 흩어진 먼지를 로봇 청소기로 직접 조작해 빨아들이고, 모든 먼지를 치우면 **Clear** 화면이 나타납니다.

## 🎯 프로젝트 설명

### 목표
- React와 Three.js 기반의 실시간 3D 렌더링을 직접 구현하며 학습
- 키보드 입력, 충돌 처리, 카메라 전환, 파티클 이펙트 등 인터랙티브 3D의 핵심 요소를 한 프로젝트에 통합
- 게임 형태의 명확한 목표(먼지 청소 → Clear)를 통해 사용자 인터랙션 흐름을 완성

### 주요 기능
- **로봇 청소기 조작**: 방향키로 청소기를 전진·회전시키며 방 안을 자유롭게 이동
  - `↑` 전진 / `←` 좌회전 / `→` 우회전
- **먼지 청소**: 청소기가 먼지에 가까워지면 자동으로 빨아들이며, 빨려 들어가는 동안 먼지 파티클과 흡입 소용돌이(vortex) 이펙트가 재생
- **충돌 처리**: 벽과 가구(책상, 의자, 침대, 화분 등)에 대한 충돌 판정으로 청소기가 통과하지 못하도록 제한
- **카메라 시점 전환**: `V` 키로 두 가지 시점을 토글
  - `방 전체 시점` — OrbitControls로 마우스 드래그하며 방 전체를 둘러보기
  - `로봇 시점` — 청소기에 카메라를 붙인 1인칭 시점
- **포스트 프로세싱**: UnrealBloomPass(블룸) 효과로 발광 오브젝트와 이펙트를 강조
- **절차적 텍스처**: Canvas API로 나무 바닥, 벽지, 패브릭 등 텍스처를 코드로 직접 생성
- **Clear & 다시하기**: 모든 먼지를 제거하면 Clear 오버레이가 나타나고, 버튼으로 먼지를 새로 배치해 재시작

## 🛠️ 기술 스택

| 분류 | 기술 |
| --- | --- |
| 프레임워크 | React 19 |
| 3D 엔진 | Three.js |
| 렌더러 | React Three Fiber (R3F) |
| 헬퍼 | @react-three/drei (`OrbitControls`, `PerspectiveCamera`) |
| 이펙트 | EffectComposer / UnrealBloomPass |
| 빌드 도구 | Vite |
| 린트 | ESLint |

## 💻 설치 & 실행

### 요구 사항
- Node.js (LTS 권장)
- npm

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
실행 후 터미널에 표시되는 주소(기본 `http://localhost:5173`)를 브라우저에서 엽니다.

### 빌드 & 미리보기
```bash
npm run build     # 프로덕션 빌드 생성 (dist/)
npm run preview   # 빌드 결과물 로컬 미리보기
```

### 린트
```bash
npm run lint
```

## 🎮 조작법

| 키 | 동작 |
| --- | --- |
| `↑` | 전진 |
| `←` | 좌회전 |
| `→` | 우회전 |
| `V` | 시점 전환 (방 전체 ↔ 로봇 시점) |

방 안의 먼지를 모두 빨아들이면 화면에 **Clear**가 표시됩니다. **다시하기** 버튼을 누르면 먼지가 새로 배치되어 다시 청소할 수 있습니다.
